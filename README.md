# Steam Achievements Exporter

A lightweight userscript for exporting Steam achievements with accurate unlocked/locked status and progress counters. Includes a floating button and a popup notification when copied to clipboard.  

---

## Features

- Copy all achievements on a Steam game achievement page  
- Automatically counts **Unlocked / Locked** achievements  
- Works on all Steam Community achievement pages

---

## Installation

1. Install a userscript manager for your browser:  

- **[Tampermonkey](https://www.tampermonkey.net/)**  
- **[Violentmonkey](https://violentmonkey.github.io/)**  

2. Go to the Userscript file on GitHub:  

[Steam Achievements Exporter](https://raw.githubusercontent.com/sypcerr/steam-achievements-exporter/main/steam-achievements-exporter.user.js)  

3. Click **Install** in Tampermonkey / Violentmonkey.  
4. Navigate to a Steam games achievement page (see below) and click the button to copy achievements.
